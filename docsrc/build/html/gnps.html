GNPS Usage¶

Using TIMSCONVERT on GNPS¶

The web version of TIMSCONVERT currently supports various data formats listed below. No installation is necessary. All that is required is a GNPS account.

  1. Upload your data to GNPS using the following instructions.

  2. Go to the TIMSCONVERT workflow page.

  3. Select your dataset.

  4. Submit your run.

  5. You will receive an email when your job has completed.
